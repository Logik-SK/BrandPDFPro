# BrandPDFPro

Professional PDF branding tool for adding headers, footers, logos, and corporate branding to PDF documents.

## Tech Stack
- Java 21
- JavaFX
- Maven
- Apache PDFBox

## Current Version
- Development: V1

## Features
- Header Templates
- Footer Templates
- Batch PDF Processing
- Page Numbering (Planned)
- Watermark Support (Planned)
